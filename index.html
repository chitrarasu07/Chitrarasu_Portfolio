Skill Set

HTML

* HTML (HyperText Markup Language) is the standard language used to create the structure of web pages.
* It defines the content and layout of a web page by using a series of elements (tags) that structure text, images, links, tables, and other multimedia content.

CSS

* CSS (Cascading Style Sheets) is a style sheet language used to control the presentation, layout, and design of HTML documents.
* It allows you to define styles for text, colors, spacing, layout, fonts, and other visual elements, ensuring consistent and aesthetically pleasing web pages.

BootStrap

* Bootstrap is a popular front-end framework developed by Twitter that helps developers quickly build responsive and mobile-first web pages.
* It provides a set of pre-designed components, templates, and tools like CSS, JavaScript, and HTML that facilitate the creation of modern web interfaces.

JavaScript

* JavaScript is a high-level, interpreted programming language primarily used to create interactive and dynamic content on websites.
* Unlike HTML and CSS, which structure and style content, JavaScript adds interactivity, functionality, and real-time updates to web pages.

React-Js

* React.js is a JavaScript library developed by Facebook for building user interfaces (UIs).
* It allows developers to create reusable and composable components to build dynamic, interactive, and scalable web applications.

Java

* Java is a high-level, object-oriented programming language developed by Sun Microsystems (now Oracle Corporation).
* It is designed to be platform-independent, meaning code written in Java can run on any device that has a compatible Java Virtual Machine (JVM).

PostgreSQL

* PostgreSQL is a powerful, open-source relational database management system (RDBMS) that uses and extends the SQL language.
* It is designed to handle complex queries and large volumes of data with high reliability, scalability, and performance.

Spring-boot

* Spring Boot is a framework for building Java applications with minimal setup.
* It provides embedded servers and simplifies dependency management.

C-Language

* C is a high-level, general-purpose programming language that was developed in the early 1970s by Dennis Ritchie at Bell Labs.
* It is considered one of the most fundamental and influential programming languages.
